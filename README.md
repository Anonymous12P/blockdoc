# BlockDoc — Blockchain-Based Document Storage System

## 📌 Overview
BlockDoc is a full-stack web application designed for secure document storage and verification using blockchain-inspired integrity tracking.  
It allows users to upload files, generate SHA-256 hashes, create blockchain-linked records, and verify whether a file has been tampered with after upload.

---

## 🚀 Features
- User Registration & Login
- Secure File Upload
- SHA-256 File Hash Generation
- Blockchain Record Creation
- Document Integrity Verification
- Activity Logging
- Dashboard Analytics
- Blockchain Explorer
- Responsive Modern UI
